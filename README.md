# Synergy FinTech Java

Учебный проект в рамках эксплуатационной практики.

## О проекте

Консольное Java-приложение для управления банковскими счетами и финансовыми операциями.

### Возможности

- Создание счетов клиентов
- Просмотр списка всех счетов
- Пополнение баланса
- Перевод средств между счетами
- Просмотр истории транзакций

## Технологии

- Java
- Объектно-ориентированное проектирование
- Коллекции (List)
- Консольный ввод-вывод
- Валидация данных

## Структура проекта

| Класс | Назначение |
| --- | --- |
| `Account` | Модель банковского счета |
| `Transaction` | Модель финансовой операции |
| `BankService` | Бизнес-логика: управление счетами и переводами |
| `Main` | Точка входа, консольное меню |

## Запуск

**Через IDE** (IntelliJ IDEA или аналогичную) -- запустить класс:

```
org.synergy.fintech.Main
```

**Через терминал:**

```bash
cd src/main/java
javac org/synergy/fintech/Main.java
java org.synergy.fintech.Main
```

## Цель

Проект иллюстрирует базовые этапы разработки ПО для финансовых технологий: анализ требований, проектирование, реализацию, тестирование, внедрение и сопровождение.
